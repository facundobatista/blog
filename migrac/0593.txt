CDPedia 0.8
<tags>

Luego de bastante laburo las últimas semanas, estoy muy contento de anunciar (de mi parte y de Python Argentina) que está en la calle la versión 0.8 de CDPedia, un proyecto que permite acceder a la información de la Wikipedia en castellano sin necesidad de una conexión a Internet. Se puede descargar libremente de la red y grabar a CDs, DVDs o memorias USB para repartirla sin restricciones.

La CDPedia funciona en cualquier computadora, ya sea que tenga Linux, MacOS o Windows como sistema operativo, y esta última versión incluye contenido de Wikipedia a Diciembre de 2012.

Para mayor información e instrucciones para la descarga, visitar <a href="http://cdpedia.python.org.ar/">la página del proyecto</a>.

<a href="http://cdpedia.python.org.ar/"><img src="http://www.taniquetil.com.ar/facundo/imgs/cdpedia.png" alt="Logo" height="300" width="300" /></a>


Esta versión trae las siguientes novedades:</p>
<ul>
<li>Renovamos completamente la generación del ejecutable para Windows, se deberían tener muchísimos menos problemas para usar la CDPedia en ese entorno.

<li>Al extraer las páginas de Wikipedia, se les hace un análisis para evitar los artículos vandalizados (en este caso, se baja una versión más antigua, pero correcta).

<li>El servidor interno ahora es multiusuario, lo que simplifica enormemente instalar CDPedia en un servidor y que sea accedida desde distintos clientes.

<li>Ahora se incluyen en los discos todas las páginas de las categorías, no sólo la primera.

<li>Actualizamos el contenido a Diciembre 2012.

<li>Varias mejoras a la hora de la generación de discos y tarballs, así como también en la calidad del código.

</ul>
<p>Si tenés discos disponibles, hacé algunas copias de CDPedia para regalar a tus familiares y amigos, a la biblioteca de tu barrio, o a la escuela de tus hijos. <strong>¡Difundí CDPedia! </strong></p> |
