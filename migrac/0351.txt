Este sábado y domingo se hace un <a href="http://wiki.python.org/moin/PythonBugDay">Global Python Sprint Weekend</a> (ya se los había comentado <a href="http://www.taniquetil.com.ar/plog/post/1/347">acá</a>). Para aquellos que quieran colaborar, la novedad es que Humitos escribió un <a href="http://humitos.wordpress.com/2008/05/05/colaborando-con-python/">documento</a> que les permitirá estar productivo en un abrir y cerrar de ojos.

¡Felicitaciones (nuevamente) a los cerebros atrás de Cocos! No sólo se mandaron una <a href="http://cocos2d.org/">linda página</a>, sino que subieron dos videos a YouTube, uno mostrando <a href="http://www.youtube.com/watch?v=mrhfaoMvpps">los efectos</a> y el otro <a href="http://www.youtube.com/watch?v=MJsd0J4G4RU">usando Cocos de forma interactiva</a>. Les recomiendo efusivamente que vean esos dos videos, ¡son buenísimos!

En función del incremento de tráfico que tuvo la lista de Python Argentina en los últimos meses, y que tenemos lectores de muchas partes del mundo hispanoparlante, quise hacer una estadística comparativa con la otra gran lista en español del planeta. Para ello recurrí a <a href="http://gmane.org/">Gmane</a>, que es el puente usado por ambas listas para tener un espejo en forma de News. Luego de pelearme durante media hora con esos números (que tienen agujeros, y están actualizados sólo hasta hace seis meses), saqué un grafiquito:
