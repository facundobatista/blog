Python 2.4
<tags>

La &uacute;ltima versión de Python ya está en la calle. Esta es una release final y estable, y se recomienda a los usuarios que actualicen a esta versión.

Python 2.4 es el resultado de casi 18 meses de trabajo sobre la versión anterior, Py2.3, y representa otra etapa en la cuidadosa evolución del lenguaje. Se mantuvieron al mínimo las nuevas características de Python, se corrigieron muchos bugs y se realizó una importante variedad de mejoras.

Entre los cambios más importantes tenemos decoradores de funciones, expresiones generadoras, mejoras al importar los módulos, y bastantes nuevos módulos (incluyendo <span style="font-style: italic">subprocess</span>, <span style="font-style: italic">decimal </span>y <span style="font-style: italic">cookielib</span>).

Para seguir investigando mire el <a href="http://www.python.org/doc/2.4/whatsnew/whatsnew24.html">What&#39;s New In Python</a> de Andrew Kuchling, los <a href="http://www.python.org/2.4/highlights.html">puntos sobresalientes</a> de esta versión, y las <a href="http://www.python.org/download/releases/2.4/notes/">notas detalladas</a> sobre Python 2.4.

Para más información y bajar los instaladores, vaya a la <a href="http://www.python.org/2.4/">página oficial</a>.[@more@] |
